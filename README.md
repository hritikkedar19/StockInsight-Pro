# 📈 StockInsight Pro

> A Professional AI-Powered Stock Market Analysis & Price Prediction Dashboard built with Streamlit, Machine Learning, Plotly, and Yahoo Finance.


---

# 🚀 Overview

StockInsight Pro is an end-to-end stock market analytics platform that provides:

- 📈 Interactive stock charts
- 📊 Technical analysis
- 🤖 AI-based next-day stock price prediction
- 📉 Trading volume analysis
- 📋 Company information
- 📅 Historical stock performance
- 💹 Buy/Sell recommendation using Machine Learning

The application downloads live stock data directly from Yahoo Finance and predicts the next day's closing price using a Random Forest Regression model.
